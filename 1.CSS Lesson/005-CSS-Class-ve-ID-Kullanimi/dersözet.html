id Kullanımı

D seçicisi, HTML üzerinde aynı id’ye sahip elemana ulaşmamızı sağlar. ID seçicisi CSS’de # ile belirtilir. ID seçicisinin kullanım amacı olarak class seçicisinden bir farkı yok diyebiliriz. İkisi de belirli HTML elementlerine CSS özellikleri eklemeye yöneliktir. Fakat id seçicisinin class seçicisinden bazı farkları vardır.
